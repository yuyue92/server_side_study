启动服务（编译、运行）：go run ./cmd

在项目根目录执行：
```
go build -o dist/blog-server.exe ./cmd   # Windows
# 或
go build -o dist/blog-server ./cmd       # mac/linux
```

完整的博客系统前端界面！这是一个纯HTML+CSS+JavaScript的单页应用，包含以下功能：

✅ 已实现的完整功能：
1. 用户认证
- ✅ 用户注册（用户名、邮箱、密码）
- ✅ 用户登录（JWT令牌管理）
- ✅ 获取用户信息
- ✅ 退出登录

3. 文章管理
- ✅ 查看所有文章列表（公开访问）
- ✅ 查看文章详情（公开访问）
- ✅ 创建文章（需要登录）
- ✅ 编辑文章（仅作者）
- ✅ 删除文章（仅作者）
- ✅ 我的文章列表

3. 评论功能
- ✅ 查看文章评论（公开访问）
- ✅ 发表评论（需要登录）

4. API测试中心
- ✅ 健康检查测试
- ✅ 认证功能测试（注册、登录、获取信息）
- ✅ 文章功能测试（创建、查询、更新、删除）
- ✅ 评论功能测试（创建、查询）
- ✅ 一键运行全部测试
- ✅ 可配置API地址

🎨 界面特点：
- 现代化设计 - 渐变背景、圆角卡片、流畅动画
- 响应式布局 - 适配各种屏幕尺寸
- 用户友好 - 清晰的提示消息、直观的操作流程
- 状态管理 - JWT令牌本地存储、用户状态显示
- 权限控制 - 根据登录状态显示不同功能

🚀 使用方法：
- 启动后端服务 - 确保Go博客API运行在 http://localhost:8080
- 打开前端页面 - 直接在浏览器中打开HTML文件

开始测试：
- 先去"API测试"页面运行全部测试，验证后端连接
- 在"注册"页面创建账号
- 在"登录"页面登录
- 发布文章、查看文章、发表评论
