# ERP/CRM 客户端 - Tauri v2 + React + Tailwind v3

## 📋 项目概述

面向中小企业的客户管理系统桌面端，集成库存管理、客户关系管理、离线缓存、扫码枪对接等功能。

## 🔧 核心功能模块

### 1. 客户管理 (CRM)
- 客户信息 CRUD
- 联系人管理
- 跟进记录
- 客户分类与标签
- 客户画像分析

### 2. 库存管理 (ERP)
- 商品档案管理
- 入库/出库操作
- 库存盘点
- 库存预警
- 扫码枪快速录入

### 3. 订单管理
- 销售订单
- 采购订单
- 订单状态跟踪
- 订单统计

### 4. 报表展示
- 销售报表
- 库存报表
- 客户分析报表
- 数据可视化

### 5. 系统功能
- 离线数据缓存
- 数据加密存储
- 企业 API 集成
- 数据同步

## 🔐 安全设计

- **数据加密**: AES-256-GCM 加密本地数据
- **密钥派生**: Argon2 派生加密密钥
- **安全存储**: 敏感信息存储在系统密钥链
- **传输安全**: HTTPS + 请求签名

## 📦 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | React 18 | UI 构建 |
| 样式框架 | Tailwind CSS 3 | 原子化 CSS |
| 状态管理 | Zustand | 轻量状态管理 |
| 图表库 | Recharts | 数据可视化 |
| 桌面框架 | Tauri v2 | 跨平台桌面应用 |
| 后端语言 | Rust | 系统级功能 |
| 本地数据库 | SQLite | 离线数据存储 |
| 加密库 | ring/aes-gcm | 数据加密 |

## 🚀 快速开始

```bash
# 安装依赖

# 开发模式
npm run tauri dev
```
